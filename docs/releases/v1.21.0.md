# Mihomo v1.21.0

`v1.21.0` 完成节点 Provider 写回的 P0 数据安全能力，并收紧侧栏、配置质量与覆写概览的视觉连续性。

## Provider 写回安全

- `proxy-providers` 使用局部 YAML 文本编辑：保留顶层顺序、未知字段、Provider 块注释、行内注释和既有缩进。
- 同名 Provider 以“来源 Profile + 名称”区分；不同来源可共存，同一 Profile 内定义不一致会在写入前阻断并列出冲突字段。
- 新增、编辑、批量导入、关联切换和删除统一先展示写入摘要；预览后 Profile 发生变化会拒绝写入，失败会恢复快照，并支持会话内单步撤销。
- 远程订阅刷新需要保留旧 Provider 时显示逐项确认；批量刷新将所有待确认项排队处理。

## 界面优化

- 侧栏列表行与主内容使用同一页面背景，降低系统侧栏材质带来的色差。
- 配置质量页各分段复用稳定的全宽内容区；字段来源支持在大量字段时滚动浏览，避免切换标签造成宽度跳变。
- 覆写概览把“顶层键/入口函数”改为全宽摘要，减少右侧无效留白。

## 无开发者账户发行说明

本版本使用 ad-hoc 签名，未经过 Apple notarization。下载、解压并移动到 `/Applications` 后，如 macOS 阻止启动，请执行：

```bash
xattr -cr /Applications/Mihomo.app
```
