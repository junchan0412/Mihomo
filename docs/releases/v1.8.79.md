# Mihomo v1.8.79

`v1.8.79` 继续收敛策略页视觉，使信息密度和交互路径更贴近参考设计。

## 主要变化

- 移除策略页旧式搜索操作条和状态横幅，让 Provider 与策略组成为页面主体。
- Provider 与策略组统一采用无描边浅灰大圆角横行，调整行高、间距和文字层级。
- 页面操作改为紧凑圆形图标按钮；模式切换继续由窗口 Toolbar 统一承载。
- Provider 行现在可进入节点详情，右侧保留独立刷新入口和更新状态。
- 策略组行保留测速、当前节点、候选数量和详情入口，GUI 编辑器仍可从页头进入。

## 验证

- 92 个 XCTest 全部通过。
- `Mihomo 1.8.79 (179)` 使用指定 Xcode-beta 构建成功。
