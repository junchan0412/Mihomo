# Mihomo v1.18.0

`v1.18.0` 聚焦规则、配置与覆写页面的信息层级和布局连续性。

## 规则

- 规则分类文本改用区分度更高的蓝、绿、青、橙、棕、紫、红等颜色映射，减少类型间的视觉混淆。

## 配置

- 配置质量的“质量总览”“字段来源”“合并层级”共享统一的内容宽度和最小内容高度，切换时保持稳定的视觉边界。
- 字段来源行支持紧凑布局：正常宽度下保留字段、来源、值和说明的并列浏览；窗口受限时自动改为纵向信息布局，避免字段过多时挤压或丢失关键信息。

## 覆写

- 覆写概览将语法状态、行数和大小保留为紧凑摘要，并让“顶层键”跨越整行。
- 顶层键摘要默认显示更多字段，并允许最多三行显示，降低右侧空白并保留完整上下文。

## 无开发者账户发行说明

本版本使用 ad-hoc 签名，未经过 Apple notarization。下载、解压并移动到 `/Applications` 后，如 macOS 阻止启动，请执行：

```bash
xattr -cr /Applications/Mihomo.app
```

首次启动核心或修复 Helper 时，macOS 会要求管理员授权。应用内更新继续通过 Ed25519 manifest、zip SHA-256、bundle identifier，以及主 App 与 Helper 的精确 CDHash 验证产物。

## 验证

- 已通过 SwiftPM 编译与测试验证。
