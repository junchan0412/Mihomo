# Mihomo v1.9.0

`v1.9.0` 是一次面向 macOS HIG、桌面交互效率与系统集成的大版本重构。

## 原生窗口与导航

- 主侧栏改为原生 Source List，使用系统 selection、Section 和 Label 语义。
- 设置迁移到标准 `Settings` Scene，`Command-,` 打开独立设置窗口。
- Profile 编辑器改为 value-based WindowGroup，可同时打开多个配置窗口。
- Toolbar 使用系统可自定义结构，路由模式位于 principal 区域；核心、系统代理与 TUN 项可移动或移除。
- 菜单新增导航、控制、网络和维护分组，并支持 `Command-1…9`、`Command-F`、`Command-R` 等快捷键。

## 连接与日志

- 连接工作区保留“最近的请求 / 活动连接 / DNS / 流量统计”四个模块，移除设备和日志簿入口。
- 连接表缩短 ID、客户端和策略列；DNS 作为连接数据的只读观测视图，不再跳转设置页面。
- 流量时间轴仅使用“直连”和“代理”两种颜色语义，修复额外颜色造成的含义不明确问题。
- 日志页重构为分类侧栏和结构化表格，支持全部、常规、网络切换与 DHCP，不添加 Mihomo 不支持的脚本类型。

## 表格、批量操作与安全交互

- AppKit 表格支持 Command/Shift 多选、Return 打开、Space 预览、Delete 删除以及多行 Context Menu。
- 连接、配置、规则、资源、日志、覆写与策略页面补齐批量操作。
- 删除、清空、批量关闭、Provider 回滚、清理网络快照和卸载组件等危险操作增加确认。
- Profile、规则和覆写的编辑操作接入 Undo/Redo；不可恢复的连接关闭继续使用明确确认。

## 辅助功能与视觉

- 页面切换和策略展开尊重 Reduce Motion。
- 卡片、边框和时间轴适配 Reduce Transparency 与 Increase Contrast。
- 字体改用语义字体；侧栏、Toolbar 和表格补齐 accessibility label、value 与 keyboard help。
- 搜索统一使用 Toolbar `.searchable`，`Command-F` 可直接聚焦当前工作区搜索框。

## 系统集成与权限

- 新增 Dock Menu，用于显示主窗口、启停核心、切换系统代理和路由模式。
- 新增 App Intents / App Shortcuts：显示 Mihomo、核心控制、系统代理、路由模式、刷新订阅与运行诊断。
- Profile 与 Provider 可通过 Spotlight 搜索并深链打开；Profile、Provider 和诊断包支持 Quick Look 或 ShareLink。
- 通知权限不再在启动时请求，只有用户主动开启订阅刷新失败通知并应用设置时才触发系统授权。

- 完成 Settings Window、Source List、Toolbar 自定义、快捷键、搜索焦点、多选表格和 accessibility tree 的真实应用 QA。
