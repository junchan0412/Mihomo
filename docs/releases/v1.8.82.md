# Mihomo v1.8.82

`v1.8.82` 修复菜单栏图标、Provider 缓存、规则状态交互、资源更新状态与设置导航问题。

## 主要变化

- 菜单栏改为固定 18×18 单色 template image，不再与动态状态文字混排。
- Provider 节点缓存可解析标准 YAML、Base64 编码订阅以及常见分享链接列表。
- 规则启用列使用原生 checkbox，取消选中即可禁用规则，重新选中即可启用。
- 资源表行支持右键“更新此资源”，双击更新行为继续保留。
- Provider 更新历史由 80 条提升至 500 条，批量更新大量资源后早期资源不会重新显示“未更新”。
- 设置成为主窗口中的应用内页面；侧栏和 `Command-,` 均切换至该页面。

## 验证

- 101 个 XCTest 全部通过，包括 YAML/Base64 Provider 缓存解析回归测试。
- 已在真实窗口确认 Provider 缓存节点数、规则 checkbox 和应用内设置导航。
- `git diff --check`、维护性审计、Release smoke test 和更新替换 smoke test 通过。
