# Mihomo v1.8.78

`v1.8.78` 聚焦策略操作路径、配置职责分层和官方内核更新。

## 主要变化

- 策略页面改为单一纵向工作区，分别展示 Proxy Provider 与策略组，减少视线跳转和多栏空间浪费。
- Provider 行提供节点数量、更新状态和刷新入口；策略组行展示当前节点、候选数量、测速和节点详情入口。
- 保留 GUI 策略组编辑器，节点详情支持选择和单节点操作。
- 覆写管理从配置页完全拆出，成为侧栏一级页面；配置页不再重复提供覆写摘要和窗口入口。
- 规则页恢复全宽表格，双击现有规则直接进入“编辑规则”。
- 默认托管内核升级至官方 MetaCubeX/mihomo `v1.19.28`，下载地址和 SHA-256 同步更新。
- 拆分策略、配置摘要和规则展示模型，减少大型 View 文件的重绘与维护负担。
