# Mihomo v1.8.84

`v1.8.84` 改进软件更新信息和菜单栏状态展示。

## 软件更新

- 更新 manifest 现在自动读取并签名完整 Release Notes。
- 软件更新窗口会展示本版本的完整更新内容，而不再只有版本标题。
- Release smoke test 会拒绝缺少完整更新日志的 manifest，防止相同问题再次发布。

## 菜单栏

- 使用与 Mihomo App icon 相似的 M 形品牌图形，保持系统菜单栏的单色适配。
- 图标右下角显示当前出站模式：规则模式 `R`、直连模式 `D`、全局模式 `G`。
- 上传与下载速率以两行紧凑布局显示。
- 用户可从菜单栏菜单或应用设置中关闭/开启速率显示，偏好会跨启动保存。

## 验证

- Swift 测试套件全部通过。
- 完成菜单栏显示、模式角标、速率开关和更新日志的真实应用检查。
- Release 包通过签名、SHA-256、bundle metadata、完整 Release Notes 与更新替换 smoke test。
