# Mihomo v1.17.0

`v1.17.0` 参考 FlClash / Sparkle 的交互密度，重点修复规则命中计数，并优化策略、规则、配置编辑、网络 / 高级工具 / 诊断 / 设置等页面的 UI 与可用性。

## 规则

- 修复规则命中计数：将控制器返回的规则类型（如 `DomainSuffix`）规范为配置写法（如 `DOMAIN-SUFFIX`），使连接命中可正确累加。
- 规则页增加类型分类筛选（域名 / IP / 地理 / 进程 / 集合 / 逻辑 / 兜底）。
- 规则表增加分类、命中、选项列，并显示总命中统计。

## 策略

- 策略页头部增加策略组 / 节点 / Provider / 核心状态摘要芯片。
- 离线预览与在线切换状态文案更清晰。

## 配置与覆写编辑

- YAML dump 默认 `allowUnicode: true`，避免中文、emoji 等被 `\uXXXX` 转义。
- 配置 YAML 与覆写编辑器统一使用带行号的高亮编辑器。
- 编辑器底部显示行数与字符数。

## 网络 / 高级工具 / 诊断 / 设置

- 文案与信息层级对齐：设置负责日常偏好，网络负责接管与 DNS，高级工具负责 Helper/Geo/备份/预览，诊断负责定位与导出。
- 各页副标题与状态摘要更明确，降低跨页职责混淆。

## 无开发者账户发行说明

本版本使用 ad-hoc 签名，未经过 Apple notarization。下载、解压并移动到 `/Applications` 后，如 macOS 阻止启动，请执行：

```bash
xattr -cr /Applications/Mihomo.app
```

首次启动核心或修复 Helper 时，macOS 会要求管理员授权。应用内更新继续通过 Ed25519 manifest、zip SHA-256、bundle identifier，以及主 App 与 Helper 的精确 CDHash 验证产物。
