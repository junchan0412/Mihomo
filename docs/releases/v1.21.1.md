# Mihomo v1.21.1

`v1.21.1` 修复节点 Provider 的重复记录与不可操作的变更预览。

## 节点 Provider

- 同名独立 Provider 与 Profile 导入 Provider 自动合并，Profile 中声明的 URL、类型、路径与更新间隔优先。
- 合并保留原有用户分组、标签和关联关系，不再在资源页显示两个相同名称的记录。
- 删除、关联和批量导入前的预览会自动清理历史重复项，并明确显示将移除的 Provider 记录数；确认按钮可正常执行变更。

## 验证

- `swift test`
- `./script/build_and_run.sh --verify`
