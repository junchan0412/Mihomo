# Mihomo v1.8.83

`v1.8.83` 聚焦状态界面的可读性、交互可靠性和资源更新时间持久化。

## 改进

- 流量时间轴新增首、中、末横轴时间标签，以 cyan、blue、indigo 区分较早到较新的时间段，并限制最多显示 48 根柱。
- 连接侧栏选中项使用更明显的 accent 背景与描边。
- 策略 Provider 的 chevron 现在可以直接展开和收起，整行点击仍然有效。
- 菜单栏图标改用 macOS 原生 monochrome Symbol，消除错误模板图片造成的黑色方块。

## 修复

- Provider 更新历史以标准化的 kind/name 键匹配，避免配置重新加载或应用重启后“最后更新”回到“未更新”。

## 验证

- Swift 测试套件全部通过。
- Release 包通过项目 smoke test，包含签名、bundle metadata、更新 manifest 与 SHA-256 校验。
