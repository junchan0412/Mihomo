# Mihomo v1.8.81

`v1.8.81` 重点改善软件更新、品牌图标、资源管理与维护类工作区的实际使用体验。

## 主要变化

- 软件更新窗口参考原生 macOS 更新器重构，提供版本摘要、Release Notes、完整性信息、跳过版本、稍后提醒和安装入口。
- 新增 light/dark App 图标、`.icns` 安装图标和单色菜单栏模板图；侧栏品牌标识同步使用新图标。
- 资源入口从引擎区移入常规区，资源批量更新使用独立的 1–12 并发设置，可在资源页和设置页调整。
- Proxy Provider 节点与策略组使用一致的页内展开和折叠动画，不再弹出独立详情窗口。
- 网络接管卡片改为自适应网格，窄窗口下保持可读。
- 高级工具增加当前 Helper 状态和分区说明，减少维护操作选择成本。
- 诊断页拆分为概览、检查结果和修复中心，避免诊断与高风险修复按钮混在同一行。
- 设置页扩大内容宽度并增加关键偏好摘要，订阅刷新并发和资源更新并发分别管理。

## 验证

- XCTest 全部通过。
- `git diff --check`、维护性审计、Release smoke test 和更新替换 smoke test 通过。
- 已对更新、资源、策略、网络、高级工具、诊断和设置完成真实窗口检查。
- App bundle 包含 light/dark PNG、菜单栏模板图与 `Mihomo.icns`。
