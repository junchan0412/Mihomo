# Mihomo v1.19.0

`v1.19.0` 聚焦桌面端导航连续性、菜单栏策略操作效率，以及节点订阅与 Profile 的解耦。

## 主窗口与菜单栏

- 主窗口左侧边栏改用与内容区相同的页面基础背景，保留原生选中态但移除突兀的侧栏底色差异。
- Menu Bar Extra 改为独立策略组面板：顶部提供全量延迟测试与状态，策略组可展开显示全部节点、当前选择和每个节点的延迟。
- 每个策略组保留单独测速入口；延迟按快速、一般、较慢和不可用状态分级展示。
- 核心、系统代理、TUN、出站模式、配置切换和诊断/更新等常用操作集中在底部控制区。

## 独立节点提供商

- 资源页新增“节点提供商”工作区；订阅 URL、更新间隔、启用状态与更新时间独立保存在 App 数据中。
- 可为当前 Profile 复选多个节点提供商；只有被选中的条目会在 runtime config 生成时写入 `proxy-providers`。
- 独立 Provider 不会改写 Profile YAML，也不会随 Profile 刷新丢失。
- 若独立 Provider 与 Profile 或覆写中的 Provider 同名，生成会停止并报告冲突，避免覆盖现有定义。

## 验证

- `swift test`：161 个 XCTest 通过。
- `./script/build_and_run.sh --verify`：SwiftPM 构建、App bundle 签名与启动检查通过。

## 无开发者账户发行说明

本版本使用 ad-hoc 签名，未经过 Apple notarization。下载、解压并移动到 `/Applications` 后，如 macOS 阻止启动，请执行：

```bash
xattr -cr /Applications/Mihomo.app
```
