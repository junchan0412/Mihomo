# Mihomo v1.8.77

`v1.8.77` 是一次面向配置语义、资源管理和 macOS 使用体验的完整重构。

## 主要变化

- 修复 Activity 的 DNS 导航，直接进入“网络 → DNS”。
- 设置改为“通用 / 远程访问 / 高级”，减少重复和认知负担。
- 网络改为“概览 / DNS / 恢复”，清晰区分 Runtime DNS 与 macOS 系统 DNS。
- 高级工具改为“运行工具 / 数据与界面 / 备份与安全 / 配置检查”，不再重复 Controller、DNS 和 Sniffer 常用设置。
- 配置列表缩短名称与来源列；覆写采用“覆写列表 + 覆写信息”双栏结构。
- 配置质量面板新增质量总览、字段来源、合并层级，并支持说明 hover 详情。
- 配置合并语义调整为：`YAML 覆写 > JS Transform > Profile 配置 > 应用默认`。
- 资源页调整列顺序和宽度，移除无关 Controller/本地解析/完成按钮；未就绪过滤保持稳定布局。
- “全部更新”同时处理远程 Provider 和本地规则集；本地资源增加存在、非空和可读性校验。
- 资源 URL 参数在界面中隐藏，避免订阅 token 暴露。
- 策略页新增 GUI 策略组编辑入口，支持节点、Provider、删除引用处理。
- 规则页增强 GUI 新建、Inspector、编辑和删除流程。

## 内部改进

- Runtime Config 不再强制删除 Profile 中的 managed keys。
- 配置字段来源统一为 YAML 覆写、JS Transform、Profile 配置、应用默认和生成结果。
- 删除资源页未使用的 Controller Provider 更新路径，统一由资源管理器更新。
- 新增本地 Provider refresh 测试和配置优先级回归测试。
- README 与开发文档按当前架构、数据流、页面职责、测试和发布流程重写。

## 安装提示

当前包使用 ad-hoc signing 且未 notarize。首次安装如被 Gatekeeper 阻止，可执行：

```bash
xattr -dr com.apple.quarantine /Applications/Mihomo.app
```

已有版本可通过应用内更新安装，客户端会验证 manifest 签名、zip SHA-256、bundle id 和 signing identifier。
