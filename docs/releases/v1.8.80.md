# Mihomo v1.8.80

`v1.8.80` 重构连接、日志、策略、规则、配置质量与覆写工作区，完善离线数据和配置作用域。

## 主要变化

- 连接工作区仅保留“最近的请求 / 活动连接 / DNS / 流量统计”四个页内分段，删除设备与日志簿入口。
- 最近请求与活动连接拆分为独立数据源；连接 ID、客户端和策略列缩短。
- DNS 改为页内连接观测表；流量统计支持策略、进程、主机名分组与多个时间窗口。
- 独立日志页改为类型侧栏和四列表格，不添加 Mihomo 不支持的脚本类型。
- Provider 缓存节点可在核心停止时显示；策略组支持图标、页内展开、隐藏策略组和不可用节点筛选。
- 规则参数（包括 `no-resolve`）与匹配值共同显示；启用状态改为紧凑符号，命中计数稳定去重并支持重置。
- 配置质量页消除等高双栏空白，优化评分、问题去重与错误/警告检查，并避免 Provider-only 配置误报。
- 覆写支持全局或指定 Profile 作用域，Runtime 构建和质量分析按相同范围过滤。
