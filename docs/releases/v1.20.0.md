# Mihomo v1.20.0

`v1.20.0` 聚焦策略选择效率，以及节点提供商与 Profile 的数据一致性。

## 策略与菜单栏

- 策略组和节点默认按可用性、延迟排序；可按未测速、快速、一般、较慢与不可用状态筛选。
- Menu Bar Extra 顶部增加节点/策略组搜索；每个策略组保留组内测速入口，展开后可查看并直接切换匹配节点。
- 当前节点与全部候选节点显示统一的延迟状态，方便在菜单栏完成快速比较和切换。

## 节点提供商与 Profile

- 节点提供商新增分组与标签，并支持按行批量导入 `名称, URL, 标签`；名称省略时自动使用 URL 主机名。
- 导入、刷新本地或远程 Profile 时，会从 `proxy-providers` 提取条目，存入节点提供商并建立 Profile 关联。
- 在资源页新增、编辑、批量导入或关联节点提供商时，变更会同步写入关联 Profile 的 `proxy-providers`。
- Profile 远程刷新时会保留旧内容中远端遗漏的 Provider，避免历史关联因上游响应不完整而丢失。
- 已同步到 Profile 的同名 Provider 不再重复注入运行时配置；`file` 类型 Provider 保持其原始类型。

## 导航

- 侧栏增加常规、引擎、应用三组可折叠分区。
- 任意侧栏入口可通过右键加入收藏，常用页面显示在顶部。
- 侧栏、页头和页脚继续使用统一的页面背景，降低与主内容区的色差。

## 验证

- `swift test`：165 个 XCTest 通过。
- `./script/build_and_run.sh --verify`：构建、签名校验和应用启动检查通过。

## 无开发者账户发行说明

本版本使用 ad-hoc 签名，未经过 Apple notarization。下载、解压并移动到 `/Applications` 后，如 macOS 阻止启动，请执行：

```bash
xattr -cr /Applications/Mihomo.app
```
