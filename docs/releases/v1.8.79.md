# Mihomo v1.8.79

`v1.8.79` 继续收敛策略、连接与日志页视觉，使信息密度和交互路径更贴近原生桌面客户端。

## 主要变化

- 移除策略页旧式搜索操作条和状态横幅，让 Provider 与策略组成为页面主体。
- Provider 与策略组统一采用无描边浅灰大圆角横行，调整行高、间距和文字层级。
- 页面操作改为紧凑圆形图标按钮；模式切换继续由窗口 Toolbar 统一承载。
- Provider 行现在可进入节点详情，右侧保留独立刷新入口和更新状态。
- 策略组行保留测速、当前节点、候选数量和详情入口，GUI 编辑器仍可从页头进入。
- 连接工作区仅保留“最近的请求 / 活动连接 / DNS / 流量统计”四个页内分段，删除设备与日志簿入口。
- 最近请求与活动连接拆分为独立数据源；连接 ID、客户端和策略列缩短，窄窗口下分段导航保持完整可读。
- DNS 改为页内连接观测表，不再跳转 DNS 设置；流量统计支持策略、进程、主机名分组与多个时间窗口。
- 独立日志页改为类型侧栏和四列表格，提供常规、网络切换、DHCP 分类，不添加 Mihomo 不支持的脚本类型。
- Provider 缓存节点现在可在核心停止时显示；策略组支持图标、页内展开，以及隐藏策略组/不可用节点筛选。
- 规则参数（包括 `no-resolve`）与匹配值共同显示，不再误作注释；启用状态改为紧凑符号。
- 配置质量页消除左右等高造成的空白，优化评分与问题去重，并避免 Provider-only 配置被误报为没有出站。
- 覆写支持全局或指定 Profile 作用域，列表会直接展示覆盖范围。
