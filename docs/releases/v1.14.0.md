# Mihomo v1.14.0

`v1.14.0` 重构连接工作区与菜单栏状态项，使高频网络观测更接近原生 macOS 客户端的紧凑交互。

## 菜单栏

- 使用原生矢量流量柱图替代旧的 Mihomo `M` 标记，避免模板图片在浅色、深色和高亮状态下显示异常。
- `R`、`D`、`G` 模式字母缩小并叠放在图标右下角，不再单独占用菜单栏宽度。
- 上传和下载速率改为无箭头的两行紧凑显示，同时为 VoiceOver 保留明确的方向语义。
- 图标与文字统一使用系统 `.primary` 颜色，自动适配菜单栏反色和不同外观。

## 独立连接工作区

- 连接从主窗口侧栏内容区拆分为独立窗口，并在主侧栏底部与菜单栏“面板”菜单提供统一入口。
- 顶部整合最近请求、活动连接、DNS 与流量统计模块、连接数量及内嵌搜索。
- 连接表格支持客户端图标，并保留紧凑 ID、时间、客户端、规则、策略、上传、下载与地址信息。
- 连接详情可折叠；“查看规则”和“定位 Provider”会返回主窗口中的对应页面。
- DNS 增加全部、本地、系统、动态分类与对应空状态说明。
- 流量统计可按策略、进程、网络适配器、设备和主机名分组。
- AppKit 表格单元格新增可选图标支持，同时维持键盘、多选与 accessibility 行为。

## 无开发者账户发行说明

本版本使用 ad-hoc 签名，未经过 Apple notarization。下载、解压并移动到 `/Applications` 后，如 macOS 阻止启动，请执行：

```bash
xattr -cr /Applications/Mihomo.app
```

首次启动核心或修复 Helper 时，macOS 会要求管理员授权。应用内更新继续通过 Ed25519 manifest、zip SHA-256、bundle identifier，以及主 App 与 Helper 的精确 CDHash 验证产物。
