# Mihomo v1.12.0

`v1.12.0` 聚焦连接活动可观察性与网络接管边界。

## 主要变更

- 连接活动在实时事件流不可用时以独立 1 秒轮询恢复，最近请求使用稳定 fallback identity。
- 连接列表 ID 改为用户可读的计数模式；历史连接详情明确显示“已结束”，不能误操作关闭。
- 菜单栏使用 Mihomo 模板图标，并将启动/停止核心合并为动态菜单项。
- 系统代理与 TUN 不再强制互斥；Helper 停止核心时可分别恢复 TUN 与系统 DNS 快照。

## 参考与验证

ClashMac 的公开仓库仅提供发布版本、README 和截图，未提供可审计的内部源码；本版本参考其实时活动与原生 macOS 状态反馈理念。Sparkle 是软件更新框架，不是连接记录实现，因此仅作为更新状态交互参考。

验证：`swift test`（128 项通过）、`git diff --check`。
