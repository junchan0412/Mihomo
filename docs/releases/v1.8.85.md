# Mihomo v1.8.85

`v1.8.85` 重绘品牌图标，并重新整理核心管理界面的职责与交互。

## 品牌与菜单栏

- App icon 重绘为艺术化纯 M 形，移除中间圆圈和双向箭头。
- light/dark 图标使用独立自适应背景，暗黑模式不再出现亮色底。
- 侧栏和更新窗口改用代码原生主题绘制，避免读取错误主题 PNG。
- 菜单栏 M 图形从不稳定的 Canvas 改为原生 Shape，修复只剩 R/D/G 字母的问题。
- R、D、G 模式角标和可选上传/下载速率继续保留。

## 概览、连接与策略

- 概览策略组增加持久化“自定义”菜单，可选择最多四个常用策略组。
- 流量时间轴只使用直连 cyan 与代理 indigo，并按时间桶显示堆叠占比。
- 连接页流量统计“名称”列从 320 缩短到 108。
- Provider 与策略组展开改用显式 snappy 动画，移除整卡隐式动画造成的跳动。

## 信息架构重构

- 网络：只负责系统代理、TUN、系统 DNS、运行时 DNS、快照和恢复策略。
- 高级工具：只负责 Helper、LaunchDaemon、数据工件、加密、备份、深链和配置预览。
- 诊断：只负责运行检查、展示结果和导出脱敏诊断包，不再重复网络修复中心。
- 设置：只保存通用、远程访问、端口、连接、Sniffer、日志和更新偏好；DNS、TUN 与恢复设置统一迁到网络页。

## 验证

- 105 项 Swift 测试全部通过。
- 完成暗黑模式图标、概览自定义、流量表列宽、策略展开和四个重构页面的真实应用 QA。
- Release 包通过签名、SHA-256、完整 Release Notes、bundle metadata 与更新替换 smoke test。
