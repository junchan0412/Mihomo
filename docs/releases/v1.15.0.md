# Mihomo v1.15.0

`v1.15.0` 重新设计菜单栏图标并统一全应用页面背景色，修复视觉一致性问题。

## 菜单栏图标

- 使用 `MihomoMarkShape`（M 字形）替代 v1.14.0 的信号柱图，恢复品牌辨识度。
- 模式字母（`R`/`D`/`G`）叠加在 M 字形右下角，使用圆形背景衬托，颜色随模式变化：
  - `R`（规则）橙色
  - `D`（直连）绿色
  - `G`（全局）蓝色
- 右侧保留两行紧凑上传/下载速率显示，可通过设置开关控制。
- 修复菜单栏速率开关与设置页 `showMenuBarTrafficRates` 不同步的问题，统一使用 `AppSettings`。

## UI 背景色统一

- 为以下 9 个主页面补齐 `MihomoUI.pageBackground`，消除与 OverviewView/LogsView 之间的背景色差异：
  - ResourcesView（资源）
  - ProfilesView（配置）
  - RulesView（规则）
  - NetworkSecurityView（网络）
  - DiagnosticsView（诊断）
  - AdvancedView（高级工具）
  - PoliciesView（策略）
  - OverridesView（覆写）
  - SettingsRootView（设置）

## 无开发者账户发行说明

本版本使用 ad-hoc 签名，未经过 Apple notarization。下载、解压并移动到 `/Applications` 后，如 macOS 阻止启动，请执行：

```bash
xattr -cr /Applications/Mihomo.app
```

首次启动核心或修复 Helper 时，macOS 会要求管理员授权。应用内更新继续通过 Ed25519 manifest、zip SHA-256、bundle identifier，以及主 App 与 Helper 的精确 CDHash 验证产物。
